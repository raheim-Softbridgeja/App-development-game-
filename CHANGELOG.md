# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2024-04-28
### Added
- Integrated `SoundManager` for interactive feedback (clicks, matches, swaps, errors).
- Implemented a "Tropical Gem" UI theme.

### Changed
- Refactored `BoardView` to render tiles with `LinearGradient` shaders for a glassy, gem-like appearance.
- Updated `colors.xml` to a vibrant tropical color palette.
- Enhanced UI components (buttons and panels) with rounded corners and modern gradients.

## [0.1.0] - 2024-04-01
- Initial project release.
- Core Match-3 game logic and engine implementation.
